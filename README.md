# InAmigos NGO Awareness Webpage 🌍

A professional, fully responsive web portal designed to spread awareness about the **InAmigos NGO** initiatives. This project demonstrates clean UI/UX design, modular coding practices, and interactive user experiences.

🚀 **[View Live Demo Here]([https://your-username.github.io/inamigos-ngo-webpage/](https://mohammadfaizanansari.github.io/inAmigos-webpage/))**

---

## 📖 About the Project
This awareness webpage was developed to showcase the core missions of the InAmigos NGO. It serves as a centralized hub for potential volunteers and supporters to learn about our impact, ongoing projects, and gallery highlights.

### Key Features:
*   **Modern Design**: Utilizes a glassmorphism-inspired UI with high-contrast color schemes for professional appeal[cite: 4].
*   **Fully Responsive**: Fluid layout that adapts seamlessly to desktop, tablet, and mobile devices[cite: 4].
*   **Interactive UI**: Includes smooth scroll animations, reveal-on-scroll effects, and hover transitions to keep users engaged[cite: 3, 4].
*   **Dynamic Elements**: Features counter animations for impact statistics and automatic year updates in the footer[cite: 3].
*   **Accessibility**: Implemented semantic HTML and descriptive `alt` tags to ensure content is accessible to all users[cite: 2, 4].

---

## 🛠️ Technical Stack
*   **HTML5**: Semantic structure for better SEO and accessibility[cite: 2].
*   **CSS3**: Custom styling, Flexbox/Grid layouts, and modern CSS variables[cite: 4].
*   **JavaScript (ES6)**: Logic for mobile navigation, scroll-triggered animations, and dynamic statistics[cite: 3].
*   **Font/Icons**: Google Fonts (Poppins) and FontAwesome[cite: 2].
